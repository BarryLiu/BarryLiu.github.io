3. 1. nginx

2.nginx停止

1. nginx -s stop 或者
2. nginx -s quit 或者
3. pkill -9 nginx

　　例如在我们的编辑环境中已经安装好了nginx,并且已启动,在命令提示符下直接输入nginx -s stop就可以停止了。

3.指令介绍

nginx -h #帮助  

nginx -v #显示版本  

nginx -V #显示版本和配置信息  

nginx -t #测试配置  

nginx -q #测试配置时，只输出错误信息  

nginx -s stop #停止服务器  

nginx -s reload #重新加载配置
